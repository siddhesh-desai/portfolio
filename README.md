# Siddhesh Desai - Portfolio

Minimal, content-first portfolio with a live neural network search widget. No frameworks, no build step.

**Live**: [siddhesh.is-a.dev](https://siddhesh.is-a.dev)

## Features

- Two-column layout (sticky sidebar + scrolling content)
- Live Neural Network inference - auto-computes TF-IDF weights from content, runs real forward propagation in-browser
- Mouse-following spotlight effect
- Archive page with full project table
- Zero dependencies, zero build step (vanilla HTML, CSS, JS + Google Fonts)

## Updating Content

All content lives in `js/data.js`. Edit that one file and push - the neural network weights auto-recompute from your content.

## Stack

HTML / CSS / JavaScript - deployed on GitHub Pages
